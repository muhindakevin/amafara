# UMURENGE WALLET - Digital Microfinance Platform

A world-class, intelligent web application for digitizing Rwanda's Ibimina (saving groups) and connecting them to banks through a secure, intelligent, and user-friendly digital microfinance platform.

## 🎨 Design Philosophy

Built with a **unique blue-and-white color theme** representing trust, innovation, and clarity. The UI features modern minimalism inspired by Claude AI's elegant simplicity.

## ✨ Features

### 🤖 AI-Powered Credit Scoring
- Analyze alternative data (mobile usage, saving history, contribution consistency)
- Suggest optimal loan options to members
- Real-time credit score assessment

### 💰 Mobile Money Integration
- Seamless connection with MTN Mobile Money and Airtel Money
- Secure deposits, withdrawals, and repayments

### 📊 Loan Management
- Members can apply for loans
- Multi-level approval workflows (Admin, Members, Cashier)
- Automatic repayment reminders
- AI-generated loan recommendations

### 💬 Notifications & Messaging
- SMS, email, and in-app notifications
- OTP authentication for secure login
- Internal group chat between members and leadership

### 📈 Analytics Dashboard
- AI-generated reports on group performance
- Loan repayment trend analysis
- Member activity insights
- Role-specific dashboards

## 👥 User Roles

1. **Member** - Register, contribute, apply for loans, chat with group
2. **Group Admin** - Approve members/loans, manage fines, send notifications
3. **Cashier** - Verify contributions, apply fines, track repayments
4. **Secretary** - Maintain records, document meetings, manage communications
5. **Agent** - Register groups, manage admins, export reports
6. **System Admin** - Full system control, manage agents, security settings

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd FrontEnd
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit:
```
http://localhost:3000
```

### Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` directory.
