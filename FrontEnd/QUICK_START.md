# 🚀 UMURENGE WALLET - Quick Start Guide

## Installation (2 minutes)

```bash
cd FrontEnd
npm install
npm run dev
```

Open: `http://localhost:3000`

## Available Routes

| Route | Description | User Role |
|-------|-------------|-----------|
| `/` | Landing Page | Public |
| `/login` | Login with OTP | All Users |
| `/member` | Member Dashboard | Members |
| `/admin` | Group Admin Dashboard | Group Admins |
| `/cashier` | Cashier Dashboard | Cashiers |
| `/secretary` | Secretary Dashboard | Secretaries |
| `/agent` | Agent Dashboard | Agents |
| `/system-admin` | System Admin | System Admins |
| `/chat` | Chat Interface | All Users |
| `/analytics` | Analytics Dashboard | All Roles |

## User Roles & Permissions

### 1. Member
- View savings and balance
- Apply for loans
- Track transactions
- Chat with group
- View notifications

### 2. Group Admin
- Approve/reject members
- Approve/reject loans
- Manage fines
- Send announcements
- Suspend/activate members

### 3. Cashier
- Verify contributions
- Apply fines
- Track loan payments
- Generate financial reports

### 4. Secretary
- Maintain member records
- Document meeting minutes
- Manage communications
- Archive documents

### 5. Agent
- Register new groups
- Assign group leadership
- Monitor performance
- Export reports

### 6. System Admin
- Manage all agents
- Control system settings
- Monitor all transactions
- Handle security
