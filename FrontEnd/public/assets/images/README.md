# Image Assets

This directory should contain high-quality images for the IKIMINA WALLET landing page.

## Required Images

### Hero Section
- `hero-community.png` - Image of Rwandan community members (men and women) sitting together in an Ibimina style meeting, saving money together
- Dimensions: 1200x800px (or similar aspect ratio)
- Style: Warm, authentic, respectful of Rwandan culture
- Colors: Soft, natural tones

### Community Story Section
- `community-savings.png` - Image of a mother from farming community using a phone for digital savings
- Dimensions: 800x600px
- Style: Modern, showing technology adoption in rural settings
- Theme: Digital financial inclusion

### Background Images (Optional)
- `pattern-1.svg` - Subtle background pattern
- `pattern-2.svg` - Alternative pattern for sections

## Image Guidelines

1. **Quality**: Use high-resolution images (minimum 72 DPI for web)
2. **Format**: PNG for photos, SVG for illustrations
3. **Optimization**: Compress images to reduce file size while maintaining quality
4. **Cultural Sensitivity**: Ensure images respect Rwandan culture and traditions
5. **Accessibility**: Include alt text descriptions for all images

## AI-Generated Images

If using AI to generate images, use prompts like:
- "A group of Rwandan men and women in traditional and modern clothing sitting in a circle, discussing savings, warm lighting, community feeling"
- "A Rwandan woman in farming clothing using a smartphone, rural setting, technology integration"
- "Bank representative meeting with Rwandan community members, professional yet warm atmosphere"
