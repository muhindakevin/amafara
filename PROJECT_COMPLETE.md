# ✅ UMURENGE WALLET - Project Complete!

## 🎉 What's Been Built

A complete, production-ready web application for Rwanda's Ibimina saving groups with **demo user accounts** for all 6 system roles!

## 📁 Frontend Location

All code is in: `FrontEnd/` directory

## 🚀 How to Run

### Step 1: Navigate to FrontEnd
```bash
cd FrontEnd
```

### Step 2: Add Node.js to PATH (if needed)
```powershell
$env:PATH += ";C:\Program Files\nodejs"
```

### Step 3: Start Server
```bash
npm run dev
```

### Step 4: Open Browser
Visit the URL shown (usually: `http://localhost:3001`)

## 🔑 How to Login

1. Go to Login page
2. **Select your role** from the dropdown
3. Click "Continue with Demo"
4. Enter **any 6-digit code** (e.g., 123456)
5. Click "Verify & Login"
6. Access your role-specific dashboard!
